# AI-Powered Education Operating System

Project documentation and planning for an AI-driven education management platform targeting learning centers in Uzbekistan.

## Quick Start

### Local Development
```bash
npm run dev
```

Visit `http://localhost:3000` in your browser.

## Deployment on Vercel

1. **Connect GitHub Repository**
   - Push this project to a GitHub repository
   - Go to [vercel.com](https://vercel.com)
   - Click "New Project" and select your repository
   - Vercel will auto-detect the configuration

2. **Deploy**
   - Vercel will automatically deploy your project
   - Your site will be live at `<project-name>.vercel.app`

## Project Structure

- `main.html` - Main project documentation and information page
- `vercel.json` - Vercel deployment configuration
- `package.json` - Project metadata

## Content

The platform documentation covers:
- Problem statement and solution
- Team roles and technology stack
- Competitive advantages
- Development roadmap (Idea → Prototype → MVP → Launch)
- Implementation phases with AI components

All content is in Uzbek language.

## Tech Stack

**Frontend:** Next.js 14, TypeScript, Tailwind CSS  
**Backend:** FastAPI, Python  
**Database:** Supabase PostgreSQL  
**Storage:** Cloudflare R2  
**Deployment:** Vercel + Railway
